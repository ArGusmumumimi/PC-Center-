## 🚀 Project Overview

PC Center คือเว็บไซต์สำหรับจำหน่ายอุปกรณ์คอมพิวเตอร์ เช่น CPU, GPU, RAM, เมาส์, คีย์บอร์ด และอุปกรณ์ไอทีอื่น ๆ

ผู้ใช้งานสามารถ:
- ค้นหาสินค้า
- ดูรายละเอียดสินค้า
- เพิ่มสินค้าลงตะกร้า
- สั่งซื้อสินค้าออนไลน์

ผู้ดูแลระบบสามารถ:
- จัดการสินค้า
- จัดการคำสั่งซื้อ
- จัดการสต็อกสินค้า
