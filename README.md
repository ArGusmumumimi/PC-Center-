# 📄 1. ข้อเสนอโครงงาน (Project Proposal)

### ชื่อโครงงาน (ภาษาไทย)

PC Center – ศูนย์รวมคอมพิวเตอร์และอุปกรณ์ไอที

### ชื่อโครงงาน (ภาษาอังกฤษ)

PC Center – Online Computer & IT Equipment Store

### หลักการและเหตุผล

ปัจจุบันการซื้อคอมพิวเตอร์และอุปกรณ์ไอทีผ่านระบบออนไลน์ได้รับความนิยมเพิ่มขึ้นอย่างต่อเนื่อง โครงงานนี้จึงพัฒนาเว็บไซต์สำหรับจำหน่ายคอมพิวเตอร์และอุปกรณ์ไอที เพื่อให้ผู้ใช้งานสามารถเลือกซื้อสินค้า ค้นหาสินค้า และสั่งซื้อผ่านระบบออนไลน์ได้อย่างสะดวก

### วัตถุประสงค์

- พัฒนาเว็บไซต์จำหน่ายคอมพิวเตอร์และอุปกรณ์ไอที
- ให้ผู้ใช้สามารถค้นหาและสั่งซื้อสินค้าได้
- ให้ผู้ดูแลระบบจัดการสินค้าและคำสั่งซื้อได้
- ศึกษาการใช้ Git และ GitHub ในการพัฒนาโครงงาน

### ขอบเขตโครงงาน

#### ผู้ใช้งาน

- สมัครสมาชิก
- เข้าสู่ระบบ
- ดูสินค้า
- ค้นหาสินค้า
- เพิ่มสินค้าลงตะกร้า
- สั่งซื้อสินค้า
- ดูประวัติการสั่งซื้อ

#### ผู้ดูแลระบบ

- จัดการสินค้า
- จัดการหมวดหมู่
- จัดการคำสั่งซื้อ
- จัดการสต็อกสินค้า
